# Controlador de Viagens

## Compilação
g++ *.cpp -o sistema

## Execução
./sistema

## Funcionalidades
- Cadastro de cidades, trajetos, transportes e passageiros
- Início e acompanhamento de viagens (avanço de horas)
- Relatórios de viagens em andamento, passageiros e transportes
